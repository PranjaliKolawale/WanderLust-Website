# WanderLust-Website
Wanderlust is a travel accommodation web application that allows users to create, view, edit, and delete property listings, making it easier to share and explore travel stays in one place. Users can enhance their listings with images and location details, helping travelers find suitable accommodations effortlessly. This features dynamic pages for browsing and viewing detailed property information, providing a seamless and organized experience.

### Features
User Authentication: Allows users to create, edit, and delete their property listings securely.
Image and Location Uploads: Users can add images and location details to enhance their listings for travelers.
Dynamic Browsing: Users can browse and view detailed property pages for each listing.
CRUD Functionality: Create, Read, Update, and Delete property listings with ease.
Centralized Platform: Helps users share and explore travel accommodations in one place.

### Tech-Stack Used:
HTML, CSS, Javascript, Bootstrap, MongoDB, Node.js, Express.js, EJS
